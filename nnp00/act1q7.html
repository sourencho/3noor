Has someone already produced an online guide to Yerevan? There are some guidebooks about Armenia and Yerevan, but we should see what information sources are available on the Web so we can learn from what we like and dislike about them.


Task: Find the URLs of sites which have tourist information about Yerevan.

Hint: Try one or more of the following.
Go to Yahoo. choose:
Regional > Countries > Armenia > Travel.
Go to http://travelguide.lycos.com or http://cityguide.lycos.com.
Go to the Lonely Planet Web site at http://www.lonelyplanet.com and search in Asia.
Go to Ask Jeeves and type: Where can I find travel information for Armenia?
Optional Competition: Can you take the material and information about Yerevan that we are finding on the Web and make a small Web site out of it? You might be able to produce a very useful online resource about Yerevan which we will put on the Web .
